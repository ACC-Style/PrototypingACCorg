# Readability Checklist & Rating Scale

Source: [Readability Evaluation Checklist and Rating Scale](https://acc-style.github.io/BrainDump/work/acc/projects/acc-redesign/discovery/rules/readability-evaluation-checklist-and-rating-scale.html)

Rate each page or section **1–5** per criterion. Sum for total (max 40).

## Criteria rubric

| Criteria | 1 Poor | 2 Below avg. | 3 Average | 4 Good | 5 Excellent |
|----------|--------|--------------|-----------|--------|-------------|
| **Reading level** | College+ / complex | 12th–college | 10th–12th | 8th–10th | 6th–8th |
| **Sentence length** | >30 words avg. | 25–30 | 20–25 | 15–20 | 10–15 |
| **Paragraph density** | Long, dense | Somewhat dense | Moderate | Brief | Concise |
| **Vocabulary** | Very specialized | Somewhat specialized | Mixed | Mostly simple | Clear, simple |
| **Jargon usage** | Excessive | Frequent, undefined | Occasional, defined | Minimal, defined | Minimal, always defined |
| **Sentence structure** | Passive, complex | Mostly passive | Mixed | Mostly active | Active, clear |
| **Scannability** | Difficult | Fairly difficult | Average | Good | Excellent |
| **CTA clarity** | Unclear | Vague | Somewhat clear | Clear | Very clear, actionable |

## Total score bands

| Total | Interpretation |
|-------|----------------|
| 7–14 | Needs significant improvement |
| 15–21 | Below average — requires revision |
| 22–28 | Average — could benefit from minor edits |
| 29–35 | Good — minor tweaks only |
| 36–40 | Excellent — highly readable |

## Supporting guidelines

| Dimension | Target |
|-----------|--------|
| Reading level | Grade-level comprehension (8th–10th for professional marketing) |
| Sentence length | **15–20 words** average for general professional audiences |
| Paragraph density | **2–4 sentences** per paragraph |
| Vocabulary | Everyday words where possible; define required clinical terms |
| Jargon & acronyms | Define on first use; drop unnecessary insider terms |
| Sentence structure | **Favor active voice** |
| Content length | Total words vs. attention span — compare peer word counts in [benchmarks.md](benchmarks.md) |

## How to audit

1. Copy page text (or per-section) into working doc — exclude nav/footer boilerplate
2. Estimate **average sentence length** (sample 5–10 sentences)
3. Score each criterion honestly against rubric
4. Note **word count** for full page or each major section
5. Compare total to [benchmarks.md](benchmarks.md) peer range
6. Present stakeholders: scores table + top 3 fixes + rewritten samples

## Scoring notes for refactored pages

| UI pattern | Scannability impact |
|------------|---------------------|
| Hub CTA cards | High — if card copy is short |
| Long intro above cards | Lowers scannability even with good UI |
| Icon lists with bold leads | High |
| Accordion hiding dense prose | Does not improve score — chunk first; accordions only for return-visit reference ([accordions.md](../refactor/accordions.md)) |
| Section hero without clear CTA label | Lowers CTA clarity |
